# Rule: 证据链溯源

- **所有结论必须绑定来源。** 每条 EvidenceChunk 的 source_pointer 必填：
  材料ID + 页码/行号/时间戳/坐标等。来源为"无来源"的结论是 bug。
- **模型不得编造卷宗中不存在的事实。** LLM 生成内容必须来自检索结果（RAG 检索到的
  片段）或知识图谱节点/边。无检索内容的生成属于越界。
- **低置信度结论只能标记为"待核实"。** final_confidence < 0.70 的内容不得进入
  报告正文的证据链结论部分，只能出现在"存疑与补证建议"段落。
- **人工复核结果必须写入图谱。** 确认/驳回/补充材料的操作都应记录在
  audit_logs 表，并更新对应的 graph_nodes / graph_edges 的 human_review_status。
- **溯源指针是完整的。** 报告中的任何结论，用户点击来源应能定位到原始材料的具体位置。
