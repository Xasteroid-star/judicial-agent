# Corpus — 参考语料库

本文件夹存放清洗过的法律原文及元数据。每条法规由一对文件组成：

```
<law_id>.md              # 清洗过的法条原文（Markdown）
<law_id>.meta.yaml       # 元数据（来源、日期、适用范围等）
```

## 当前语料

| law_id | 名称 | 发布机关 | 施行日期 | 条数 |
|--------|------|----------|----------|------|
| `criminal_procedure_evidence` | 刑事诉讼法·第五章 证据 | 全国人大常委会 | 2018-10-26 | 15条 (50-64) |
| `exclusion_illegal_evidence` | 严格排除非法证据规定 | 两高三部 | 2017-06-27 | 42条 |
| `electronic_data_rules` | 电子数据收集提取审查判断规定 | 两高一部 | 2016-10-01 | 30条 |

## 约定

- **法条原文来自官方来源，绝不靠 LLM 生成。** LLM 仅用于清洗格式。
- **每条法规的 `.meta.yaml` 必须通过 `LawMetadata` Pydantic 模型校验。**
- **添加新法规 = 放一个 `.md` + 一个 `.meta.yaml`，代码零改动。** Loader 自动发现全部文件。
- **不硬编码法规名称、法域或文件名** — 检索和报告逻辑对 N 条法规通用。
